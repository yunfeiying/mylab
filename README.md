# Highlighti Mobile PWA (Distribution)

This folder contains the production-ready build of the Highlighti Mobile App.

## Deployment
Upload the contents of this folder to any static web hosting service:
- **GitHub Pages**
- **Netlify**
- **Vercel**

## Usage
1. Open the URL on a mobile device.
2. Use "Add to Home Screen" to install as a PWA.
3. Import your backup JSON from the PC extension.

## Files
- `index.html`: Main entry point (renamed from mobile.html).
- `mobile-core.js`: Core logic (Routing, Rendering).
- `mobile-sync.js`: Data Import/Export/WebDAV.
- `modules/`: Database and Storage utilities.
